
=====================================================
                TIC-TAC-TOE GAME
=====================================================

Description:
This program is a simple 2-player Tic-Tac-Toe game
played in the console. Players take turns entering
row and column coordinates to place their mark
(X or O) on a 3x3 board.

Features:
- Interactive game board display
- Input validation (prevents invalid or occupied moves)
- Win detection (rows, columns, diagonals)
- Draw detection when the board is full
- Replay option after each game
- Instruction screen before gameplay

How to Play:
- Player 1 uses 'X'
- Player 2 uses 'O'
- Enter moves using row and column numbers (0–2)
  Example: 1 2 places your mark in row 1, column 2
- First player to get 3 in a row wins
